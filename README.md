# Logistik - namafolder

Sistem logistik berbasis web untuk mengelola stok barang, barang masuk, dan barang keluar.

## Fitur
- CRUD Barang
- Pencatatan Barang Masuk
- Pencatatan Barang Keluar
- Laporan Stok

## Teknologi
- Laravel
- MySQL
- Bootstrap

## Cara Menjalankan Proyek (Windows & Linux)

### Prasyarat
- PHP 8.x
- Composer
- MySQL
- Git
